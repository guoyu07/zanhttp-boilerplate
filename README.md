# zanhttp-boilerplate
Youzan ZanPhp HTTP project Boilerplate

## 如何使用
1. 安装swoole扩展，编译参数：
`--enable-async-mysql --enable-async-httpclient --enable-sockets --enable-async-redis`

2. 安装composer

3. 在根目录下执行composer update

4. 配置php.ini
```ini
zanphp.RUN_MODE = test
zanphp.DEBUG = true
```

5. 启动http server
`php bin/httpd`

6. 访问`127.0.0.1:8030`
